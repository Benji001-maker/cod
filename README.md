# 💥 Call of Duty: Mobile (IDLE Edition)

A text-based, realistic battle simulator built in Python that brings the intense gunplay mechanics of Call of Duty: Mobile straight to your command terminal.

## 🎮 Features
* **Custom Weapon Loadouts:** Choose between the balanced M4 Assault Rifle, the high-damage DL Q33 Sniper, or the rapid-fire Fennec SMG.
* **Dynamic Combat Mechanics:** Features critical headshot multipliers, tactical sliding to lower enemy tracking accuracy, and realistic ammo management.
* **Scorestreaks:** Build up points during battle to unlock and call in a UAV for enhanced accuracy!
* **Visual Health Bars:** Clean rendering of player and enemy status directly in the interface.

## 🚀 How to Run
1. Make sure you have **Python 3** installed on your system.
2. Download or clone this repository.
3. Open your terminal or command prompt, navigate to this folder, and run:
   ```bash
   py cod_sample.py
